# slop-cleaner
A discord theme with less distractions

## Usage:
- Install Vesktop [https://github.com/Vencord/Vesktop]
- Turn on the light theme in Settings -> Appearance
- Place the css file in the themes folder (found in the vesktop settings).
- Turn on the theme and enjoy!

## My personal recommendations:
### Settings -> Accessibility
- Change "Text size in chat" to 14px
- Enable "Always underline links"
- Set "Space between message groups" to 0px
